H2SWF

H2SWF is a Flash and JavaScript script based tool inspired by sIFR to create custom font headlines. H2SWF is written as a plugin to jQuery and uses SWFObject to embed the SWF font files. Download Source

H2SWF has been developed for: Firefox 2 and 3, IE 6,7 and 8, Safari 3 and 4. It degrades gracefully for non JavaScript environments.

$('.blocking-1').h2swf({
	alpha : .2,
	blocking : [6, 10, 2, 28],
	leading : 1,
	tracking : -2,
	background_color : '000000',
	color : '333333',
	font_size : 32,
	pad_desc : 3,
	swf : "../flash/header.swf",
	height: 'callback'
});
